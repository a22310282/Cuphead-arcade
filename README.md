## CONTROLES CUPHEAD:
>W = SUBIR
>S = BAJAR
>A = MOVER IZQUIERDA
>D = MOVER DERECHA

## CONTROLES MUGMAN:
>W = SUBIR
>S = BAJAR
>A = MOVER IZQUIERDA
>D = MOVER DERECHA




## CONTROLES HILDA
>FLECHA IZQUIERDA Y FLECHA DERECHA (solo se mueve en horizontal, en las coordenadas de la antepenúltima viga)

>G = DISPARA BOMBA (solo se mueve en vertical)

## PARA COMPILAR
En la terminal, escribir "make runmain"
